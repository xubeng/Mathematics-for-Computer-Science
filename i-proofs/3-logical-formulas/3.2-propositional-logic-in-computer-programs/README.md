# 3.2 计算机程序中的命题逻辑

命题和逻辑联系一直存在于计算机程序中。例如，考虑下面的代码片段，它可能是C、C++、Java的一种：

                                                             if  \(  x &gt; 0 \| \|  \(x  &lt;=  0  &&  y  &gt;  100\)  \) 

                                                                  .

                                                                  .

                                                                  .

                                                                  _（更多的指令）_

Java 使用符号 \| \| 表示“或”，符号“&&”表示“与”。
